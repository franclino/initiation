# INITIATION App

## What this is
A spiritual/archetypal journey app (iOS + Android) built with React Native / Expo.
8 seasonal archetypes based on the Celtic Wheel of the Year.
Monetization: 36€ single archetype / 222€ full wheel / 19€ deepening.

## Stack
- React Native with Expo SDK 54
- expo-router (file-based routing)
- Fonts: Montserrat (headings), Quicksand (body)
- Backend planned: Supabase
- Payments planned: RevenueCat
